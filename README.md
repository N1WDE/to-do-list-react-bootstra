# To Do List - React Bootstrap

## 📌 Descripción

Aplicación web tipo To-Do List para gestionar tareas y metas personales. Permite visualizar tareas con fecha límite en una interfaz moderna y responsive.

## 🛠️ Tecnologías utilizadas

* React
* React Bootstrap
* CSS

## 🚀 Cómo ejecutar el proyecto

### Opción 1: PlayCode

1. Ir a: https://playcode.io/react
2. Copiar el código del proyecto
3. Ejecutar

### Opción 2: Ejecutar en local

1. Clonar el repositorio:
   git clone https://github.com/TU-USUARIO/todo-react-bootstrap.git

2. Entrar a la carpeta:
   cd todo-react-bootstrap

3. Instalar dependencias:
   npm install

4. Ejecutar el proyecto:
   npm start
